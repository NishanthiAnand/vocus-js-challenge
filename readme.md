# Welcome to the Vocus programming exercise

* You'll need Node.js and NPM installed on your machine.

## API Installation
1. `npm install`
2. `npm run start`
3. Navigate to `localhost:3000`

## Client-side Installation
1. `npm install`
2. `ng serve`
3. Navigate to `localhost:4200`
